# UI Screens Specification

## Design Reference

**CRITICAL**: All UI implementation must follow the design system defined in:
- `@.claude/skills/frontend-design.md`

**Core Principles**:
- Notion-inspired: clean, spacious, typography-focused
- Warm neutrals, subtle shadows
- Generous whitespace
- Smooth micro-animations
- Drag-and-drop interactions
